## Video 2: Working with Spacing Properties (4 minutes)

### Learning Objectives
- Master padding and margin visual feedback
- Understand measurement overlays
- Learn keyboard shortcuts

### Script

**[00:00-00:15] Opening**

> "Welcome back! In this video, we'll dive deep into spacing properties - padding, margin, and gap - and how visual feedback makes them easy to master."

**[00:15-01:00] Padding Properties**

> "Let's start with padding. When I hover over the padding control, watch what happens..."

*[Show: Hover triggers green overlay]*

> "The green overlay shows the padding area - the space between the content and the border. The measurement lines show exactly how many pixels. Currently, it's 16 pixels on all sides.
>
> Now watch as I change it to 24 pixels..."

*[Show: Type 24, watch animation]*

> "See how smoothly it animates? You can watch the space grow in real-time. This makes it easy to judge if 24 pixels is too much, too little, or just right."

**[01:00-01:45] Margin Properties**

> "Margins work the same way, but with an orange color. Orange indicates external spacing - the space outside the element."

*[Show: Hover over margin]*

> "Here's a pro tip: use consistent spacing values. Try 8, 16, 24, or 32 pixels. This creates visual rhythm. The measurement lines help you maintain that consistency across your entire email."

**[01:45-02:30] Gap Properties**

> "For flex and grid layouts, there's the gap property. This controls spacing between child elements."

*[Show: Gap property with multiple elements]*

> "Watch how the measurement lines show the gap between each element. This is perfect for button groups, image galleries, or any layout with multiple items."

**[02:30-03:15] Keyboard Shortcuts**

> "Speed up your workflow with keyboard shortcuts:
>
> - **Tab** to move to the next property
> - **Up/Down arrows** to adjust values
> - **Enter** to apply
> - **Escape** to cancel
>
> The visual feedback works with keyboard navigation too!"

*[Show: Demo of keyboard workflow]*

**[03:15-03:45] Advanced Tips**

> "Here are some advanced tips:
>
> 1. **Compare before and after** - Note the measurement before changing
> 2. **Use the same scale** - 8px, 16px, 24px, 32px
> 3. **Watch the animation** - It helps you visualize the change
> 4. **Check neighboring elements** - Make sure spacing is consistent"

**[03:45-04:00] Closing**

> "Next video: colors and visual properties. See you there!"

### Recording Checklist

- [ ] Start with messy spacing
- [ ] Show measurement lines clearly
- [ ] Demonstrate smooth animations

## Video 5: Tips, Tricks & Troubleshooting (4 minutes)

### Learning Objectives
- Discover hidden features
- Learn troubleshooting techniques
- Understand accessibility features
- Master keyboard workflows

### Script

**[00:00-00:15] Opening**

> "Welcome to the final video in this series! Today, I'm sharing tips, tricks, and troubleshooting advice to make you a visual feedback power user."

**[00:15-01:00] Hidden Features**

> "Let me show you some hidden features you might have missed:
>
> **1. Quick Property Discovery** - Hover over any property to learn what it does without changing anything.
>
> **2. Animation Speed** - The animation speed varies by property type. Spacing is fast (150ms), colors are smooth (200ms).
>
> **3. Overlay Transparency** - Overlays are semi-transparent so you can see the content underneath.
>
> **4. Off-Screen Indicators** - If a component is outside the viewport, you'll see an indicator arrow."

*[Show: Each hidden feature]*

**[01:00-02:00] Keyboard Ninja**

> "Become a keyboard ninja with these shortcuts:
>
> - **Tab/Shift+Tab** - Navigate properties
> - **Up/Down Arrows** - Adjust values by 1
> - **Shift + Up/Down** - Adjust by 10
> - **Enter** - Apply and close
> - **Escape** - Cancel changes
> - **Cmd/Ctrl+Z** - Undo
>
> Visual feedback works perfectly with keyboard navigation!"

*[Show: Fast keyboard workflow demo]*

**[02:00-02:45] Accessibility Features**

> "The system is built with accessibility in mind:
>
> **1. Reduced Motion** - In your browser settings, enable 'Reduce motion' and watch - no animations, just instant feedback. Perfect if you're sensitive to motion.
>
> **2. Keyboard Navigation** - Everything works with just a keyboard.
>
> **3. High Contrast** - Works with high contrast modes.
>
> **4. Screen Readers** - Property changes are announced."

*[Show: Accessibility features demo]*

**[02:45-03:30] Troubleshooting**

> "Having issues? Here's how to fix them:
>
> **Visual feedback not showing?**
> - Check if a component is selected
> - Make sure you're hovering over a property
>
> **Animations too fast/slow?**
> - They're optimized, but you can enable reduced motion
>
> **Performance slow?**
> - Close other tabs
> - Reduce canvas complexity
>
> **Overlays blocking view?**
> - They're transparent - move your mouse to clear them"

**[03:30-03:50] Best Practices**

> "Here are my top 5 best practices:
>
> 1. **Hover before editing** - Preview what will change
> 2. **Use consistent values** - 8, 16, 24, 32 pixels
> 3. **Watch animations** - They help you judge the result
> 4. **Keep it simple** - Don't overuse complex properties
> 5. **Check mobile** - Visual feedback works on all breakpoints"

**[03:50-04:00] Closing**

> "That's it! You're now a visual feedback master. Build better emails faster. Thanks for watching, and happy designing!"

*[Show: End card with resources]*

## Technical Recording Setup

### Equipment

**Minimum:**
- 1080p screen recording
- USB microphone
- Quiet recording environment

**Recommended:**
- 4K screen recording (export 1080p)
- Condenser microphone with pop filter
- Audio interface
- Second monitor for notes

### Software Recommendations

**Screen Recording:**
- **OBS Studio** (Free, professional)
- **Loom** (Easy, web-based)
- **Camtasia** (Feature-rich, paid)
- **ScreenFlow** (Mac, professional)

**Video Editing:**
- **DaVinci Resolve** (Free, professional)
- **Adobe Premiere Pro** (Professional, paid)
- **Final Cut Pro** (Mac, professional)
- **iMovie/Movie Maker** (Simple, free)

**Audio Editing:**
- **Audacity** (Free)
- **Adobe Audition** (Professional)
- **Logic Pro** (Mac)
